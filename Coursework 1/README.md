The code can be ran by executing 1D_steady_state_convection_diffusion.py file

If there are any missing packages in the environment, run "pip install -r requirements.txt" in the terminal

By default, the Global and Local Peclet numbers are 5.0 and 0.5, respectively

The program will output three (3) figures for each of the corresponding schemes (CDS, UDS, PLDS), and depending on the IDE, it may show them one at a time in a single window

To accomodate the above, an .ipynb file is included to show all three figures at once
